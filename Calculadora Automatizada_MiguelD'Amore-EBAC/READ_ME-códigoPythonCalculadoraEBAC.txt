O código do arquivo CalculadoraEBAC.py contém comentários explicativos
sobre o que o sistema oferece e como oferece. 

A calculadora funciona num sistema de Funções que são gerenciadas pela
Função "menu()"


if menuOpcao == 1:
      adicao()
      break
      
    elif menuOpcao == 2:
      subtracao()
      break

    elif menuOpcao == 3:
      multiplicacao()
      break

    elif menuOpcao == 4:
      exponenciacao()
      break

    elif menuOpcao == 5:
      mediaSimples()
      break


De acordo com a opção escolhida pelo input do usuário, o menu chama 
a função correspondente.

O código conta com a Função "limparTela()" que utiliza a biblioteca 'os' 
para garantir uma experiência mais limpa ao usuário pelo terminal!
